Build for Android:
`gradle :mobile-app:assembleDebug`

Run on iOS
Run project `ios-app/iosApp.xcodeproj`

Run on Desktop
`gradle :desktop-app:run`

Build for Wear OS
`gradle :wear-app:assembleDebug`
